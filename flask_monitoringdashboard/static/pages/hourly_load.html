Information
Graph

You can hover the graph with your mouse to see the actual values. You can also use the buttons at the top of the graph to select a subset of graph, scale it accordingly or save the graph as a PNG image.

Axes

The X-axis presents a number of days. The Y-axis presents every hour of the day.

Content

The color of the cell presents the number of requests that the application received in a single hour. The darker the cell, the more requests it has processed. This information can be used to validate on which moment of the day the Flask application processes to most requests.
